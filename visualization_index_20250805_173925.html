📖 使用说明

这个可视化工具展示了多智能体协作框架中不同智能体之间的对话流程，包括：

🔧 System Prompt
显示每个智能体的系统提示，定义其角色和能力边界
👤 User Prompt
展示用户请求和智能体接收到的具体任务描述
⚙️ 工具调用
可视化工具调用过程，包括参数、执行结果和状态
🤖 Assistant Response
显示智能体的最终响应和任务完成情况

💡 调试建议：通过观察工具调用的成功/失败状态，可以快速定位问题所在。特别关注Verilog设计智能体是否尝试调用禁止的testbench相关工具。
